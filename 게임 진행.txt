게임진행

게임start →상황설명 → 튜토리얼진행 (탐사선의 기본에너지소비, 기체 에너지 투자 (업그레이드 ), 탐사원식량분배) → 플레이어의 튜토리얼 실습  → 튜토리얼 진행 2 (탐사할 행성리스트의 정보값 읽는법 )→ 플레이어의 튜토리얼실습 (행성택1) → 튜토리얼 3 (탐사선의 현재 정보 [항해 일수, 현 탐험행성 수+현 탐사선 상태(선택, 이동(이동 정도), 도착&도착&탐사), 기체 업그레이드 상황. 자원량 ]확인 방법 명령어 + 하루종료 명령어) → 하루 종료 명령어 실습 통한 Day1 종료

Day 2 시작 → 플레이어의 식량 북배와 탐사선의 기본 에너지 소비, 기체 에너지 투자 과정 진행 → 튜토리얼4(행성 이동 중의 상담 시스템에 관하여) → 플레이어 실습 → 플레이어가 원하는 만큼 상담 진행(최소 0~최대4회, 1인 최대 1회*4) → ~ → 플레이어의 자율 하루 종료 → Day 2 종료

Day3 시작 → ~탐사 행성 도착까지 반복~

Day n 시작 → 탐사행성 도착 → 튜토리얼5 (행성 정보 읽기, 탐사대원 선택) → 플레이어 실습 → 튜토리얼6 (탐사진행 방법-실습과 함께 진행-, 탐사 종료 후 탐사일지 읽기, 행성 떠나기 명령어) → 모든 튜토리얼 종료, 플레이어의 자율 플레이 시작
